# Project Images

Place your project images in this folder.

## Image Guidelines

- Recommended size: 1200x600px or similar aspect ratio
- Format: JPG, PNG, or WebP
- File size: Keep under 500KB for optimal performance

## Naming Convention

Use descriptive names like:
- `buy-the-best.jpg`
- `wasteless-app.png`
- `pacemaker-ai.jpg`
- etc.

Then update the image path in `src/components/Projects.js` for each project.
